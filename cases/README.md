# 案例库

真实运行的 TSC 案例。

**案例是整个框架最重要的贡献类型。**  
理论可以在书桌上写，案例必须在现实中活。

---

## 提交案例

```bash
cp -r cases/_template cases/你的TSC名称
# 填写文件后提交 PR
```

数据不完整没关系。有任意真实数据的案例，比没有任何案例更有价值。

→ [查看案例模板](./_template/CASE_TEMPLATE.md)
