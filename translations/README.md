## 参与翻译

1. 开 Issue 认领目标语言（防止重复劳动）
2. 在 `translations/[语言代码]/` 目录下创建翻译文件
3. 翻译优先级：README → 创世层模板 → 白皮书v3.0
4. PR 标题：`[translation][语言代码] 翻译描述`

**翻译说明**：
- 技术术语（TSC、CT、VSM等）保留英文缩写，首次出现时注明全称
- 法典内容翻译时，保持法律文本的准确性高于流畅性
- 如有翻译争议，以中文原文为准

---

## Contributing a Translation

1. Open an Issue to claim your target language
2. Create files in `translations/[language-code]/`
3. Priority: README → Genesis Layer template → v3.0 Whitepaper
4. PR title: `[translation][lang-code] description`
