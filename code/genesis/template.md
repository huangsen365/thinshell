# 创世层模板

> 创世层是 TSC 法典的最高层。在 TSC 存续期间**不可更改**。
>
> 填写时请慢慢来——这些内容将在 TSC 整个生命周期里保持不变。

---

## 使用方式

1. 复制本文件到你的 TSC 私有文档
2. 填写所有 `[在此填写]` 标记的部分
3. 与所有核心成员共同确认内容
4. 记录签署时间（截图/电子签名/链上均可）

**写作提示**：
- 描述"为谁创造什么价值"，而不是"做什么事情"
- 避免描述具体产品或服务（那些可以随时改变）
- 问自己：如果这件事完全实现了，这个 TSC 还有理由存在吗？

---

## 第二条：不可违背原则

无论任何情况，本 TSC 不得：

**2.1** 以任何成员认为合理的理由，故意对非恶意第三方造成重大伤害。

**2.2** 向客户或合作方提供已知为错误的关键信息。

**2.3** 限制成员的退出自由，或对退出行为实施惩罚（合理结算义务除外）。

**2.4** 允许任何单一成员永久控制法典修改权或核心财务权。

**2.5** 消灭合法的反对意见，或报复表达反对意见的成员。

> 以上五条为默认条款，**不可删除**。可在下方添加本 TSC 特有的不可违背原则：

**2.6** [可选，本 TSC 特有原则]

---

## 第三条：基本成员权利

每一位以任何类型加入本 TSC 的成员，享有以下不可剥夺的权利：

- **知情权**：了解与自己贡献和收益相关的完整信息
- **发声权**：在任何关于 TSC 运作的讨论中表达意见，不受压制
- **退出权**：完成当前承诺的任务后，无惩罚退出
- **声誉携带权**：退出后，可携带并使用在 TSC 内建立的声誉记录
- **提案权**：提出规则修改提案，须在合理时间内获得正式回应

---

## 第四条：消亡条件

以下任一条件成立时，本 TSC 应启动解散程序：

**4.1** 存在宣言中的核心意图已完全实现，且无法产生新的有意义意图。

**4.2** TSC 已持续违背第二条中任一不可违背原则超过 90 天，且无法恢复合规。

**4.3** 持有超过 75% 治理权的成员一致同意解散。

**4.4** [可选，本 TSC 特有的消亡条件]

---

## 第五条：解散时的资产处置

TSC 解散时，剩余资产按以下顺序处置：

1. 优先结清所有成员的未结算贡献款项
2. 偿还 TSC 对外的任何债务
3. 将 TSC 的公共知识贡献到 TSC 生态知识公地（如适用）
4. 剩余资产按各成员的贡献比例分配
