# 质量管控规则集

---

### QC-001：AI 输出外部交付前人工审查

**触发**：AI 生成内容准备对外发送前  
**规定**：人类成员确认后方可发送  
**建议有效期**：12 个月

---

### QC-002：关键建议须附置信度说明

**触发**：AI Agent 给出影响资金或客户关系的建议时  
**规定**：AI 必须同时给出置信度评估和主要不确定性来源  
**建议有效期**：6 个月

---

### QC-003：关键数据双模型交叉验证

**触发**：输出涉及可验证的具体数据（数字·日期·人名·引用）  
**规定**：用第二个不同模型交叉验证，不一致时人工核查  
**建议有效期**：6 个月
