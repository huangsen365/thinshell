# 新成员入驻规则集

---

### OB-001：接入决策 48 小时响应

**触发**：新成员提出加入申请  
**规定**：核心委员会 48 小时内给出明确回复，不得无限期搁置  
**建议有效期**：12 个月

---

### OB-002：新成员第一周陪跑

**触发**：新成员正式加入后第一周  
**规定**：指定一名现有成员为联系人，24 小时内响应新成员问题  
**建议有效期**：12 个月

---

### OB-003：新 AI Agent 试用期

**触发**：新 AI Agent 加入 TSC  
**规定**：前 2 周为试用期，每日人类成员抽查输出质量，达标后正式激活  
**建议有效期**：12 个月
