## 规划中的合约

### 1. 多方分账合约（split-payment）

**用途**：项目完成后，自动按预设比例将款项分发给多个成员钱包。

**目标链**：Solana（手续费低，速度快）

**核心功能**：
- 部署时设置分账比例（各成员地址 + 比例）
- 收款后触发自动分账
- 比例验证（不符则拒绝执行）

**参考**：0xSplits（Ethereum 上的多方分账协议，可参考其逻辑在 Solana 上实现）

---

### 2. CT 注册合约（ct-registry）

**用途**：链上记录各成员的 CT 数量，提供不可篡改的治理历史。

**关键设计**：CT **不可转让**，合约需强制执行这一点。
