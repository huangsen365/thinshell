# 行为准则 / Code of Conduct

**简短版：直说。尊重。真实。认领。**

---

## 我们鼓励的

- **直说**：不喜欢某个设计就说，给理由，不需要客气
- **尊重**：批评想法，不攻击人，对事不对人
- **真实**：分享失败和局限，比漂亮理论更有价值
- **认领**：说了要做就做，做不了就说，让别人来
- **异见**：反对意见是礼物，提出异见不受惩罚

## 我们不接受的

- 人身攻击、骚扰、任何形式的歧视
- 压制反对意见
- 利用本项目从事欺骗性商业活动

## 冲突处理

1. 当事人先直接沟通
2. 无法解决时在 Issue 里公开描述
3. 核心维护者给出有约束力的意见
4. 任何一方可以选择退出，不受惩罚

---

*"任何人都可以提出反对意见且不受惩罚"——此条不可通过 RFC 修改。*

---

## Code of Conduct (English)

**Short version: Be direct. Be respectful. Be honest. Follow through.**

We welcome disagreement. We do not welcome personal attacks.

Anyone can express opposition without penalty. This clause is non-negotiable.
